This is Impress.js

Created by Bartek Szopka

He thought it would be fitting to make a open source presentation tool.

That would more than give Prezi a run for its money.

Lets start assuming some Base code. You have used html before right?

To begin we need to have a standard html with a body's open and closing tags and impress.js right before the closing tag.

From there add a wrapper in the body with a ID of "impress". From there out you can run any container element with a id of "step" to make your presentation.

After that begins the fun stuff. Using inline, jquery or javascript coding you can then add in element data. Such as for position, data-x ,-y and -z. All of the transition have been manipulted useing data- changes.

Even 3D effects. Use similar syntax. This has a data-rotate- x/y set to it.

These can be combined to make some pretty anoying transitions.

So Mayby I didnt explain it the best. But I sure had fun and I know you will with impress.js!

As with all things new and exciting tools have your shivs ready for cross browser compatability.
